## 🔑 Купить ключ

Показывает список тарифов:

| Кнопка | Тариф | Цена |
|--------|-------|------|
| ⚡ 1 месяц | 30 дней | 170 ₽ |
| 📅 3 месяца | 90 дней | 450 ₽ (−12%) |
| 📅 6 месяцев | 180 дней | 720 ₽ (−29%) |
| 📅 1 год | 365 дней | 1200 ₽ (−41%) |
| ◀️ Назад | Закрыть меню тарифов | — |

После выбора тарифа бот автоматически:
1. Берёт свободный ключ из пула
2. Отправляет `.conf` файл
3. Показывает дату истечения

---

## 👤 Личный кабинет

Показывает все активные ключи пользователя.

**Для каждого ключа отображается:**
- IP адрес сервера
- Тариф и дата покупки
- Дата истечения
- Сколько дней осталось + прогресс-бар

**Кнопки:**

| Кнопка | Что делает |
|--------|-----------|
| 📥 Ключ #N (IP · дней осталось) | Скачать `.conf` файл этого ключа |
| 🛒 Купить ещё ключ | Открыть выбор тарифа для нового ключа |
| 💬 Поддержка | Открыть чат с поддержкой |

> Можно иметь несколько ключей одновременно — например, один для телефона, другой для ноутбука.

## Команды (вводятся вручную)

| Команда | Кто может | Что делает |
|---------|-----------|-----------|
| `/start` | Все | Приветствие и главное меню |
| `/mystatus` | Все | Открыть личный кабинет |
| `/help` | Все | Открыть инструкцию |
| `/admin` | Только админ | Открыть панель администратора |
| `/revoke <telegram_id>` | Только админ | Отозвать все ключи пользователя |

---

## ⚙️ Панель администратора (`/admin`)

| Кнопка | Что делает |
|--------|-----------|
| 📊 Статистика | Кол-во ключей в пуле, активных пользователей, подписок |
| 👥 Пользователи | Список всех активных пользователей с датами истечения |
| 📥 Добавить ключ | Инструкция как добавить `.conf` файл в пул |
| 🗑 Отозвать ключи | Список пользователей — кликаешь, отзываешь все ключи |
| ◀️ Назад | Вернуться в главное меню панели |

### Как добавить ключ в пул (через бот)

Просто **отправь `.conf` файл** в личку боту — он автоматически добавит его в пул.
Бот ответит: `✅ Ключ добавлен! 📦 Свободных: N`

### Уведомления администратору

Бот автоматически присылает уведомления при:
- 🆕 Новой покупке (имя, тариф, IP, дата истечения)
- ❌ Истечении подписки пользователя

---

## 🔔 Автоматические уведомления пользователям

Бот сам пишет пользователю когда:

| Событие | Сообщение |
|---------|-----------|
| Подписка истекает через 3 дня | ⚠️ «Подписка истекает через N дн.» |
| Подписка истекла | ❌ «Одна из твоих подписок истекла» |
| Ключ отозван админом | ❌ «Твои подписки деактивированы» |

Проверка происходит каждые **12 часов** автоматически.
